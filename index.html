data-exp : HTML의 커스텀 데이터 속성 중 하나

진혁의 경험치

data-exp는 "exp"라는 이름의 사용자 정의 데이터 속성

값은 "2025-09-01"이라는 문자열

브라우저엔 안 보이지만, 자바스크립트로 꺼내 쓸 수 있다

const div = document.querySelector('div');

console.log(div.dataset.exp); // "2025-09-01"

변수 div에 저장된 값은 DOM 요소 객체이고 그 객체 안에 객체면서 프로피티인 dataset이 있다

exp는 dataset의 프로퍼티다 -> 닷으로 접근 가능

DOM 요소 객체 -> js가 태그 하나하나를 객체로 인식
